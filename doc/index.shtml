Bei OBanToo handelt es sich um eine Bibliothek mit Tools für das Banking. Zur Zeit sind folgende Klassen realisiert:

  • DTAUS-Parser
    DTAUS-Dateien parsen für die Weiterverarbeitung. Seit Version 0.3 werden auch mehrere logische Dateien pro physikalischer Datei unterstützt. Ab Version 0.8 kann optional eine Fehlertoleranzgrenze angegeben werden.
  • DTAUS-Writer
    DTAUS-Dateien mit einer oder mehreren logischen Dateien erstellen. Pro logischer Datei können entweder Überweisungen oder Lastschriften eines Kontos enthalten sein. Ein Mixed-Inhalt ist nicht zulässig.
  • DTAUS2PDF
    Ausgabe von DTAUS-Dateien im PDF-Format. Die Klasse kann stand-alone oder innerhalb einer Applikation genutzt werden. Hinweis! Für die PDF-Generierung wird iText benötigt. iText kann hier bezogen werden. Das iText.jar muß sich im Classpath befinden.
  • QIF-Parser
    QIF-Dateien parsen.
  • QIF-Writer
    QIF-Dateien erstellen

OBanToo wird innerhalb von Hibiscus und JVerein eingesetzt. Die GWVS Gesellschaft für Wirtschafts- und Verwaltungsservice mbH, Regensburg setzt ebenfalls auf OBanToo.

OBanToo benötigt minimal JDK 1.4 und steht unter LGPL.
